一款跨平台音乐播放器工具 · 支持鸿蒙/iPhone/Android/windows/mac/linux

## ✨ 项目简介

"拼好歌"移动端 是一款面向技术爱好者的个人跨平台音乐播放器工具，支持鸿蒙/iPhone/Android/windows/mac/linux设备使用。项目兼容洛雪音乐音源插件脚本生态（小部分不兼容）。项目是借鉴洛雪音乐源码编写出来的。应用提供完整的播放控制、歌单同步、定时关闭、后台播放完、暗黑模式等功能，界面简洁无广告。核心优势在于打破了iOS平台App必须上架应用市场的限制、鸿蒙系统无法安装Android App的限制。无需越狱、无需签名、无需TestFlight。项目代码开源且免费，如你是付费使用本项目，建议申请仅退款

## 🚀 如何使用

> iOS/华为纯血鸿蒙客户端只能在微信小程序使用，且微信小程序端不打算上架微信小程序应用市场，需要用户自行下载微信开发者工具编译代码后用手机扫码使用。Android/windows/mac/linux可以直接下载客户端使用。

使用的关键流程为：下载/搭建客户端 → **选择服务器模式** → 导入自己/第三方的音源插件脚本。

### 🎯 服务器模式选择（v1.0.10 新增）

首次打开 APP 会弹出手动选择服务器模式：

| 模式 | 适合人群 | 说明 |
| :--- | :--- | :--- |
| **专属服务器** | 已部署自己服务器的用户 | 使用自己的服务器，支持多端同步、AI 推荐等全功能 |
| **公共服务器** | 没有服务器的用户 | 由社区分享者提供公共服务器节点，导入本地音源插件即可播放 |

> 模式可在 设置 → 服务器地址 / 公共服务器节点 中随时切换。

### 📋 使用步骤

#### Android/windows/mac/linux设备

可以直接下载安装 APK，无需配置开发环境。

1. **下载安装拼好歌**
   - 访问 [GitHub Releases](https://github.com/erikjamesgz/phg-music/releases) 下载最新版本 APK
   - 安装到手机即可使用
2. **选择服务器模式**
   - 首次打开 APP 会自动弹出模式选择弹窗
   - **专属服务器模式**：填入服务器地址 `https://你的域名/你的API_KEY`，点击测试
   - **公共服务器模式**：无需配置服务器，直接导入本地音源插件即可使用
   > 如果没有自己的服务器，可以使用拼好歌的附属服务器项目：[关联项目（服务器二选一）](https://github.com/erikjamesgz/phg-music#-%E5%85%B3%E8%81%94%E9%A1%B9%E7%9B%AE%E6%9C%8D%E5%8A%A1%E5%99%A8%E4%BA%8C%E9%80%89%E4%B8%80)
3. **导入音源插件**
   - 在设置页面点击"音源插件管理"
   - 点击"在线导入"
   - 输入音源插件脚本地址
   - 点击"导入"
   - 选择默认调用音源插件
   > 建议导入 2-3 个音源插件，当某个插件不可用时将自动切换其他插件获取链接

#### iOS/华为纯血鸿蒙 设备

iOS 和纯血鸿蒙系统用户需要使用微信开发者工具编译项目。

##### 第一步：准备环境

1. **下载项目代码**
   - 准备一台电脑
   - 下载本项目代码到本地
2. **注册微信小程序开发者**
   - 注册成为微信小程序开发者
   - 获取 AppID（此 AppID 不能分享小程序给他人使用，每个用户需要自己注册微信小程序开发者获取自己的 AppID）
   - 教程：[微信小程序快速开始](https://developers.weixin.qq.com/miniprogram/dev/framework/quickstart/getstart.html)
3. **安装微信开发者工具**
   - 下载并安装微信开发者工具
   - 打开项目并设置 AppID

##### 第二步：配置项目

1. **配置开发者工具**
   - 在微信开发者工具中配置"不校验合法域名"
   - 教程：[配置教程](https://blog.csdn.net/weixin_66708577/article/details/156834305)
2. **生成预览二维码**
   - 点击微信开发者工具右上方的"预览"按钮
   - 生成二维码供手机扫描

##### 第三步：手机端配置

1. **开启开发者模式**
   - 手机扫描二维码打开小程序
   - 开启小程序开发者模式
   - 教程：[开启开发者模式](https://blog.csdn.net/m0_46613429/article/details/124845091)
2. **选择服务器模式**
   - 打开拼好歌小程序的设置页面
   - 首次使用会自动弹出模式选择弹窗，也可在设置中手动切换
   - **专属服务器模式**：点击"添加服务器地址"，填入 `https://你的域名/你的API_KEY`，点击测试
   - **公共服务器模式**：无需配置服务器，直接导入本地音源插件即可使用
   > 如果没有自己的服务器，可以使用拼好歌的附属服务器项目：[关联项目（服务器二选一）](https://github.com/erikjamesgz/phg-music#-%E5%85%B3%E8%94%94%E9%A1%B9%E7%9B%AE%E6%9C%8D%E5%8A%A1%E5%99%A8%E4%BA%8C%E9%80%89%E4%B8%80)
3. **导入音源插件**
   - 在设置页面点击"音源插件管理"
   - 点击"在线导入"
   - 输入音源插件脚本地址
   - 点击"导入"
   - 选择默认调用音源插件
   > 建议导入 2-3 个音源插件，当某个插件不可用时将自动切换其他插件获取链接

### ⚠️ 重要说明

#### 关于音源插件脚本

- ❌ **本项目不提供任何音源插件脚本**
- ❌ **不提供音源插件脚本的修改和定制服务**
- ✅ **支持洛雪音乐音源插件脚本生态（小部分不兼容）**
- ⚠️ **部分混淆过的脚本不支持，后续会持续优化**
- 💡 **公共服务器模式下，音源插件仅保存在本地，不会上传到服务器**
- 💡 **建议导入 2-3 个音源插件，播放时自动切换可用插件获取链接**

#### 关于服务器版本兼容性

- 客户端会自动检查服务端版本兼容性，版本过低时弹窗提醒升级
- 最低要求服务端版本：**v1.0.10**

#### 如何获取音源插件脚本

建议通过以下方式获取：

- 百度搜索"洛雪音乐音源插件脚本"
- 谷歌搜索"洛雪音乐音源插件脚本"
- GitHub 搜索"洛雪音乐音源插件脚本"
- 参考洛雪音乐音源插件脚本的编写规范自己编写

#### 常见问题

- **导入后无法播放**：该音源插件脚本在拼好歌项目中不支持，请更换其他音源插件脚本
- **播放异常**：部分混淆过的脚本不支持，建议使用未混淆的脚本
